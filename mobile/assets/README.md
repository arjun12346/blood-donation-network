# Mobile Assets

This folder should contain the following image/sound assets:

- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen (1242x2436)
- `adaptive-icon.png` - Android adaptive icon foreground (1024x1024)
- `favicon.png` - Web favicon (512x512)
- `notification-icon.png` - Notification icon (96x96)
- `notification-sound.wav` - Notification sound

Note: Placeholder files need real images for production build.
